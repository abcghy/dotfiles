# vim commentary
## shortcut
- `gcc` comment current line
- select and `gc` comment the selected text

# nvim tree
## shortcut
- `tt` toggle tree
- `tf` tree focus

## shortcut
- `<space>ff` search files in directory

- `<space>gg` show lazygit in vim
